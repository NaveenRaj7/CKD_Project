The project contains code (in R language) written for the classification/identification of Chronic Kidney Disease (CKD) cases. 

The dataset can be downloaded [here](https://archive.ics.uci.edu/ml/datasets/chronic_kidney_disease).

To run the project:

1. Open R Studio.
2. Execute command, getwd(). This shows the present directory the user is currently in.
3. Extract all the files in the zip folder to the current directory.
4. Open the .rmd file in R Studio and click the ‘knit’ button.
